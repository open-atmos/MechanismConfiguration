Mechanisms#

Mechanisms are collections of reactions and have the following format:

camp-data:
  - name: my mechanism
    type: MECHANISM
    reactions:
      - type: RXN_ARRHENIUS
        ...
      - type: RXN_TROE
        ...

{
  "camp-data": [
    "name": "my mechanism",
    "type": "MECHANISM",
    "reactions": [
      {
        "type": "RXN_ARRHENIUS",
        ...
      },
      {
        "type": "RXN_TROE",
        ...
      }
    ]
  ]

The name is required and must be unique. Mechanisms with the same name will be merged into a single mechanism. The type is also required and must be MECHANISM. The reactions should be an array of Reactions.
